# Awards Platform Research Kit

This folder contains an implementation-ready user research package for an awards management platform serving Admins, Judges, and Entrants.

## Files

1. `interview-guides.md`
- Moderated interview scripts for all 3 personas.
- Includes screening, warmup, core probes, and wrap-up.

2. `usability-test-script.md`
- Task-based usability protocol for Admin, Judge, and Entrant flows.
- Includes success criteria, severity scoring, and observer notes.

3. `hypothesis-survey.md`
- Quant survey to validate hypotheses H1-H6.
- Includes item bank, response scales, and analysis mapping.

4. `analysis-template.md`
- Synthesis template for findings, confidence scoring, and decisions.
- Includes hypothesis verdict matrix and prioritization model.

5. `participant-screener.md`
- Recruiter-ready screener for Admin, Judge, and Entrant participants.
- Includes quotas and role-specific qualification rules.

6. `session-scoreboard-template.md`
- Live usability logging schema with severity and confidence scoring.
- Includes task catalog and daily rollup metrics.

7. `decision-memo-template.md`
- Evidence-to-decision template for release planning.
- Includes options analysis, KPI forecast, and risk/mitigation sections.

## Suggested execution order

1. Run screening + schedule interviews.
2. Conduct interviews (n=8 per persona minimum).
3. Run usability tests (n=6 per persona minimum).
4. Send quant survey (n>=100 mixed persona responses where possible).
5. Fill `analysis-template.md` and produce decision memo.

## Minimum evidence thresholds

- Confirmed hypothesis: appears in >=60% of target persona participants and is supported by behavioral evidence (not just opinion).
- Rejected hypothesis: contradictory evidence in >=40% of target persona participants.
- Inconclusive: mixed evidence or insufficient sample.

## Deliverables this kit supports

- Persona cards with jobs, mental models, trust signals.
- Competitive teardown and flow-level frictions.
- Hypothesis validation outcomes.
- Impact x effort prioritization.
- End-to-end journey maps with emotional highs/lows.
- Anti-pattern list for design and product guardrails.
